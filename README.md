# Java Marksheet Management System

A simple console-based Java application developed as part of my Semester 1 coursework.

## Original Development Date
February 2025

## About the Project

This project generates a student marksheet using user input from the console. It accepts student details and subject marks, calculates the percentage, assigns grades, and displays the final marksheet in a formatted manner.

## Features

- Student information input
- Subject marks entry
- Percentage calculation
- Grade generation
- Formatted marksheet display
- Console-based user interface

## Technologies Used

- Java
- Scanner Class
- Conditional Statements
- Loops
- Console Formatting

## How to Run

1. Download or clone the repository.
2. Open the project in any Java IDE or editor.
3. Compile the Java file.

```bash
javac marksheet.java
```

4. Run the program.

```bash
java marksheet
```

Enter Marks of Subject 3: 88
